**Archivo:** `src/emailService.js`  
**Líneas:** 36-48


> Sí, la validación y sanitización con `express-validator` (incluyendo `.escape()`) se aplica en `src/app.js` antes de que los datos lleguen al servicio de correo. Agregué un comentario en el código para hacerlo explícito:
> ```javascript
> // Nota: los datos ya fueron sanitizados con express-validator en app.js
> ```

**Resolución:** ✅ Resuelto – Se agregó comentario aclaratorio.
