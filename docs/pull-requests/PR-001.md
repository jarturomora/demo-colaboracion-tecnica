## 📋 Descripción

Esta pull request establece la estructura base del proyecto **Formulario de Contacto**. Incluye la configuración inicial de Node.js con Express, el esqueleto del formulario HTML, la ruta del servidor para procesar el envío, y el servicio básico de correo electrónico con Nodemailer.

Es el punto de partida del proyecto y define la arquitectura que se usará en las siguientes iteraciones.
