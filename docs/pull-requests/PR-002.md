## 🎨 Decisiones de Diseño

### Variables CSS
Se definieron variables CSS personalizadas en `:root` para facilitar la personalización del tema:

```css
:root {
  --color-primario: #2563eb;
  --color-primario-oscuro: #1d4ed8;
  --radio-borde: 8px;
  --sombra: 0 4px 16px rgba(0, 0, 0, 0.08);
}
```

### Metodología BEM
Se adoptó BEM (Block-Element-Modifier) para evitar conflictos de estilos:
- `.campo` → bloque
- `.campo__etiqueta` → elemento
- `.campo__entrada--textarea` → modificador

### Mobile-first
Los estilos base están diseñados para pantallas pequeñas, y se usan `@media` queries para ampliar el diseño en pantallas mayores a 600px.
