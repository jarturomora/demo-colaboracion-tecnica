## 🔍 Cambios Técnicos Destacados

### Arquitectura del servicio
Se separaron las responsabilidades en funciones específicas:

```
emailService.js
├── crearTransportador()     → Crea el transportador SMTP
├── enviarCorreoContacto()   → Función principal de envío
├── generarTextoPlano()      → Genera el cuerpo en texto plano
└── generarPlantillaHTML()   → Genera la plantilla HTML
```

### Campo Reply-To
```javascript
replyTo: `"${nombre}" <${correo}>`,
```
Esto permite que el equipo responda directamente al remitente sin copiar manualmente su correo.

### Transportador para pruebas
En el entorno de pruebas (`NODE_ENV=test`), se usa el transportador `jsonTransport` de Nodemailer que no envía correos reales pero permite verificar los datos del mensaje:

```javascript
if (process.env.NODE_ENV === 'test') {
  return nodemailer.createTransport({ jsonTransport: true });
}
```
